# Браузерная игра Memory

### Об игре

Суть игры состоит в том, чтобы найти одинаковые карты

### Информация

Игра была написана в рамках изучения курса на Udemy по Phaser.js

## Курс: https://www.udemy.com/course/phaser-3/
---
Установка приложения
```sh
npm install
```

Запуск приложения
```sh
npm run dev
```
